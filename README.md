# Basic HTML Project

A simple multi-page website built with semantic HTML only.

## GitHub Repository

- Repository URL: https://github.com/txsunshine722/basic-html-project

## Project Goal

This project demonstrates how to build a clean, accessible, multi-page website using only semantic HTML.

## Pages Included

- homepage.html: Main landing page with an introduction and about section.
- articles.html: A list of articles and resources that inspired the project journey.
- projects.html: A portfolio-style list of projects with external links.
- contact.html: Contact details and a basic HTML form (name, email, phone).

## HTML Approach

- Uses semantic structure: header, nav, main, section, article, footer.
- Uses form semantics: label, input, button.
- Uses no CSS styling files and no JavaScript.

## SEO Basics Included

Each page includes foundational SEO tags in the head:

- title
- meta description
- meta robots
- canonical link

## How to View Locally

1. Clone the repository:

   git clone https://github.com/txsunshine722/basic-html-project.git

2. Open the project folder.
3. Open homepage.html in your browser.
4. Use the navigation links to visit all pages.
